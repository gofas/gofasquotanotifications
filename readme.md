# Gofas Notificações de Cota para WHMCS

Módulo addon WHMCS que monitora o uso de disco e banda dos serviços de hospedagem e envia alertas automáticos por email ao atingir limites configurados.

## Requisitos

- WHMCS 8.x
- PHP 8.x
- cPanel com acesso à API
- ionCube Loader
